一、Quantumult简介：什么是Quantumult？

Quantumult是由QUANTUM核心团队开发的一款专业代理工具，主要面向iOS系统用户。它支持HTTP、SOCKS5等多种代理协议，能够帮助用户绕过网络限制，访问被屏蔽的网站和服务。Quantumult支持高度自定义的配置，使用户能够根据个人需求调整代理节点、流量规则和脚本功能，极大提升网络访问的自由度和体验。

Quantumult的核心功能

  • 多重代理模式
    用户可灵活添加并管理多条代理服务器线路，满足不同访问需求和不同网络环境下的切换。

  • 灵活规则配置
    通过域名、关键词等规则判断流量走向，实现部分流量直连，部分流量走代理，节省资源同时提升速度。

  • 节点测速
    内置测速功能，帮助用户快速判断各代理节点的实时性能，方便选择最快的节点。

  • 脚本支持
    支持JavaScript脚本，扩展更多自动化功能，如节点自动更新、测速自动执行等。

三、Quantumult配置文件详解

Quantumult的配置文件通常以quantumult.conf为扩展名，是一份文本文件，包含多个功能模块，主要分为以下几部分：

3.1 节点配置

节点即代理服务器的详细信息，包括服务器地址、端口和协议类型，是Quantumult运行的基础。

示例：

ini
[Geometry]
name = Example US Server
type = http
hostname = us.example.com
port = 8080

  • name：节点名称，用于识别和选择。

  • type：协议类型，如http、socks5。

  • hostname：服务器域名或IP地址。

  • port：服务端口。

3.2 规则配置

规则定义了不同流量如何处理，常见有“直接连接”与“代理连接”两种策略。

示例：

pgsql
[Rule]
DOMAIN-SUFFIX, example.com, Proxy
DOMAIN, example.net, Direct

  • 通过规则，指定访问example.com的流量走代理服务器。

  • 访问example.net则直接连接，不经过代理。

这类规则使用户可以灵活控制网络访问路径，提高访问效率，减少不必要的代理资源浪费。

3.3 脚本设置

Quantumult支持用户编写或导入JavaScript脚本，来扩展功能，比如节点自动切换、节点测速、自动更新配置等。

示例：

csharp
[Script]
// 此处添加用户自定义脚本代码

脚本部分提升了Quantumult的自动化能力，使网络管理更为智能。

FAQ

Mellow 是否支持自定义 DNS？
支持。Mellow 的配置文件中可设置 DNS 服务器地址及模式（如 DoH、DoT）。用户可使用安全 DNS，如 Cloudflare 1.1.1.1 或 Google DNS，以减少污染风险。
VLESS 的 XTLS 模式能提升传输效率吗？
XTLS 模式通过减少冗余加密和握手次数，降低延迟，提高吞吐量。在高并发和短连接场景下，效率明显优于普通 TLS，同时保持数据安全性和隐蔽性。
Sing-box 的 route 功能如何工作？
Sing-box 提供强大的路由系统，允许用户基于 IP、域名、GEO 数据库、进程名等条件精细分流。配置完成后，程序自动匹配规则，将不同流量转发到对应的出口。
Nekobox 如何在 Android 上导入配置？
打开 Nekobox，点击“导入配置”按钮，选择订阅链接或本地 JSON 文件。程序会解析后自动生成节点列表，用户可一键连接或测速筛选最优节点。
Clash 的 Proxy Group 能实现哪些策略？
Proxy Group 可以定义多种策略，如按延迟选择、随机、轮询或手动选择节点。结合分流规则，可实现针对不同网站或应用的智能代理，提高网络体验。
Potatso Lite 节点手动切换有什么优势？
手动切换节点可以让用户根据网络延迟和用途选择最合适的节点，适合在节点不稳定或特殊访问需求下使用。虽然不自动，但可以灵活调整，提高访问效率。
Mellow 如何通过配置实现多出口分流？
Mellow 可在配置文件中定义多个 outbound，并结合 routing 规则实现按域名、IP 或策略组分流。这样不同类型的流量可走不同节点，提高使用体验。
Potatso Lite 是否可以导入多个节点？
支持。用户可手动输入或扫描二维码导入多个节点，并可修改或删除节点信息。虽然不支持自动订阅更新，但足以满足轻量用户对节点管理和分流需求。
Potatso Lite 可以设置自定义 DNS 吗？
支持自定义 DNS，可配置本地或远程 DNS 服务器，包括 DoH/DoT。正确配置可提升解析速度、防止 DNS 污染，提高访问稳定性和安全性。
QuantumultX 与 Shadowrocket 相比的最大区别是什么？
QuantumultX 在规则系统和自动化脚本上更灵活，支持丰富的 rewrite 与 filter 功能，而 Shadowrocket 则更轻便易用，适合普通用户快速切换节点。二者可根据使用习惯与功能需求选择。
